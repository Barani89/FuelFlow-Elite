⛽ Smart Fuel

🚀 Overview

Smart Fuel is a modern and responsive frontend website for a Smart Petrol Bunk & EV Charging Station. The project is designed with a clean user interface using HTML, CSS, and JavaScript.

✨ Features

- Responsive Design
- Modern Navigation Bar
- Hero Section
- Petrol Services
- EV Charging Services
- Online Booking Form
- Contact Page
- Smooth Animations
- Mobile Friendly UI

🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
